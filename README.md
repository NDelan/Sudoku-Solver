# SUDOKU SOLVER

Java-based graphical Sudoku Solver for solving and visualizing Sudoku puzzles.

## 🚀 Running Sudoku:
To run Sudoku on the file Sudoku 

```sh
javac Sudoku.java
java Sudoku
```
